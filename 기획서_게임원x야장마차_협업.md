# 게임원 × 야장마차 협업 커뮤니티 기획서

## 1. 프로젝트 개요

### 1.1 프로젝트명
**게임원 엘리트야구 커뮤니티** — 야장마차 콘텐츠 연동 플랫폼

### 1.2 협업 당사자
| 구분 | 스탯앤코 | 오늘부터아마야구 |
|------|----------|------------------|
| 플랫폼 | 게임원 (야구 기록 플랫폼) | 야장마차 (엘리트야구 콘텐츠) |
| 역할 | 커뮤니티 플랫폼 제공, KBSA 데이터 연동 | 콘텐츠 제작 및 연재 |
| 강점 | KBSA 유지보수 관리자 (데이터 열람 가능) | 11년 운영 경력, 팔로워 3.2만, 게시물 7,335개 |

### 1.3 목표
- 엘리트 야구 커뮤니티 신규 런칭 및 활성화
- 야장마차 콘텐츠의 꾸준한 게재를 통한 회원 유입
- 게임원 커뮤니티 게시판 내 엘리트야구 게시판 신설 및 동시 연재
- KBSA 경기일정/결과 데이터를 실시간으로 작성자에게 제공

### 1.4 타겟 사용자
- **1차 타겟**: 엘리트 야구 선수의 학부모 (초·중·고)
- **2차 타겟**: 현역 엘리트 야구 선수
- **3차 타겟**: 아마추어 야구 팬 및 스카우트 관계자

## 2. 현황 분석

### 2.1 오늘부터아마야구 현황
- 1인 회사, 11년째 엘리트 야구 콘텐츠 운영
- 인스타그램·페이스북 기반 콘텐츠 연재
- 팔로워 3.2만, 게시물 7,335개의 활성화된 커뮤니티
- 광고 수입 기반 운영 (피드 내 광고란 편성)
- 직접 기록지·영상을 찾아보며 콘텐츠 제작 (1인 체제의 한계)

### 2.2 콘텐츠 유형 (야장마차 기준)
| 콘텐츠 유형 | 설명 | 사용자 반응 |
|-------------|------|-------------|
| 경기 결과 카드 | 스코어, 주요 선수, 이닝별 기록 | 높음 |
| MVP 선정 | 경기별 MVP 선수 선정 및 소개 | 매우 높음 (선수·학부모 열광) |
| 리그 순위표 | 권역별 순위 및 승점 현황 | 보통 |
| 경기 일정표 | 주말리그, 대회 일정 안내 | 높음 |
| 선수 하이라이트 | 특별 기록 달성 선수 피처링 | 매우 높음 |
| 야장마차 뉴스 | 종합 뉴스 형태의 이슈 정리 | 높음 |

### 2.3 핵심 인사이트
- 본인의 이름이 MVP로 거명되는 것에 대한 선수·학부모의 강력한 반응
- 학부모 커뮤니티 특성상 정보 공유 욕구가 높음
- SNS 기반의 한계: 검색·아카이빙·데이터 연동의 어려움
- 웹 커뮤니티로의 확장 시 광고 수익 모델 다변화 가능

## 3. 커뮤니티 플랫폼 구성

### 3.1 사이트맵

```
홈 (메인)
├── 야장마차 콘텐츠
│   ├── 경기 결과 피드
│   ├── MVP 선수 피처링
│   ├── 야장마차 뉴스
│   └── 선수 하이라이트
├── KBSA 경기 센터
│   ├── 경기 일정
│   ├── 경기 결과
│   ├── 리그 순위
│   └── 대회별 보기 (주말리그, U-15, 전국대회 등)
├── 엘리트야구 게시판
│   ├── 자유게시판
│   ├── 학부모 게시판
│   ├── 선수 정보 게시판
│   └── 대회/입시 정보
├── 회원
│   ├── 회원가입 / 로그인
│   ├── 마이페이지
│   └── 알림 설정
└── 광고 영역
    ├── 배너 광고
    └── 피드 내 네이티브 광고
```

### 3.2 메인 페이지 구성
1. **히어로 영역**: 최신 야장마차 콘텐츠 슬라이더
2. **실시간 경기 결과**: KBSA 데이터 기반 당일/최근 경기 결과
3. **MVP 하이라이트**: 최근 선정된 MVP 선수 카드
4. **인기 게시글**: 엘리트야구 게시판 인기글
5. **경기 일정 위젯**: 이번 주 경기 일정 미리보기
6. **광고 배너**: 스폰서 광고 영역

### 3.3 핵심 기능

#### A. 야장마차 콘텐츠 허브
- SNS에 게시되는 야장마차 콘텐츠를 웹에 동시 게재
- 카드 뉴스 형태의 비주얼 콘텐츠 뷰어
- 경기별/대회별/선수별 필터링 및 검색
- SNS 공유 버튼 연동

#### B. KBSA 경기 데이터 연동
- 스탯앤코의 KBSA 유지보수 권한을 활용한 실시간 데이터 연동
- 작성자(오늘부터아마야구)에게 경기 일정·결과를 즉시 제공
- 콘텐츠 작성 시 데이터를 자동 참조할 수 있는 작성 도구
- 데이터 연동 항목: 경기 일정, 스코어, 이닝별 기록, 선수 기록

#### C. 엘리트야구 게시판 (게임원 연동)
- 게임원 커뮤니티 게시판 내 엘리트야구 전용 게시판 신설
- 야장마차 콘텐츠 동시 연재
- 학부모·선수 간 소통 공간
- 대회 정보, 입시 정보 공유 게시판

#### D. 회원 시스템
- 간편 회원가입 (SNS 로그인 연동)
- 관심 학교/선수 팔로우 기능
- 경기 결과·MVP 선정 시 푸시 알림
- 학부모/선수/일반팬 회원 구분

## 4. 작성자 지원 시스템 (KBSA 데이터 활용)

### 4.1 콘텐츠 작성 워크플로우 개선

**현재 (AS-IS)**
```
기록지 수동 검색 → 영상 확인 → 수기 데이터 정리 → 콘텐츠 제작 → SNS 업로드
```

**개선 후 (TO-BE)**
```
KBSA 데이터 자동 수집 → 작성자 대시보드 표시 → 콘텐츠 템플릿 자동 생성 → 편집/보완 → 웹+SNS 동시 게재
```

### 4.2 작성자 대시보드 기능
- 오늘의 경기 일정 한눈에 보기
- 경기 결과 실시간 업데이트
- 선수별 주요 기록 하이라이트 자동 추출
- 콘텐츠 작성 템플릿 (경기 결과 카드, MVP 카드 등)
- 게시 이력 및 조회수 통계

## 5. 수익 모델

### 5.1 광고 수익
- 배너 광고: 상단/사이드/하단 배너
- 네이티브 광고: 피드 내 자연스럽게 삽입되는 광고
- 야장마차 콘텐츠 내 스폰서 영역 (기존 SNS 광고 모델 계승)

### 5.2 추가 수익 기회
- 프리미엄 회원 서비스 (상세 선수 데이터 열람)
- 야구용품 업체 제휴 광고
- 대회/캠프 정보 유료 홍보

## 6. 기대 효과

### 6.1 오늘부터아마야구 측
- 1인 체제의 데이터 수집 부담 경감
- SNS 외 웹 플랫폼으로의 채널 확장
- 콘텐츠 아카이빙 및 검색 가능성 확보
- 광고 수익 모델 다변화

### 6.2 스탯앤코 (게임원) 측
- 엘리트야구 커뮤니티라는 신규 사용자 풀 확보
- 3.2만 팔로워 기반의 초기 회원 유입 효과
- KBSA 데이터의 가치 극대화

### 6.3 사용자 측
- 엘리트 야구 정보의 원스톱 플랫폼
- 경기 결과·MVP 정보를 빠르게 확인
- 학부모·선수 간 커뮤니티 소통 공간
